Accounts

  • POST

    /login

    Log in to an existing account.

    • user

      — Username or email of the account trying to be logged in.

    • password

      — Password of the account trying to be logged in.

  • POST

    /signup

    Register a new account.

    • user

      — Username of the account to be registered.

    • email

      — Email of the account to be registered.

    • password

      — Password of the account to be registered.

  • POST

    /session

    Get a representation of a session.

    • user

      — Username or email of the account trying to get the session from.

    • password

      — Password of the account trying to get the session from.

  • PUT

    /update/username

    Modifies the username of an existing account.

    • user

      — Username or email of the account to update.

    • password

      — Password of the account to update.

    • new-user

      — New username.

  • PUT

    /update/email

    Modifies the email of an existing account.

    • user

      — Username or email of the account to update.

    • password

      — Password of the account to update.

    • new-email

      — New email.

  • PUT

    /update/password

    Modifies the password of an existing account.

    • forgotten

      — Whether the operation is because of a forgotten password or not.

    • user

      — Username or email of the account to update.

    • password

      — Password of the account to update.

    • new-password

      — New password.
