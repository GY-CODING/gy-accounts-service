2.0.0

BREAKING VERSION.
Deprecation and delete of the old authentication service, now the service is based on Auth0 and the use of JWT. Achievements metadata systems are now delegated to this service, instead of the old approach of delegating it to the Fall of the Gods API.

Now, GY Accounts supports authentication with Email/Username and Password or with Google (redirecting you to our Google form page).

Also, new exception management got implemented in this version, in order for it to be more scalable and maintainable.

1.1.0

Internal migration from MySQL database to MongoDB. Also, different key management, now the email remains the only primary key for the user accesses.

1.0.1

Expanded the services to include updates for the username, email, password and a forgotten password service.

1.0.0

The service is released with login, sign up and session services.

Public documentation is provided.
